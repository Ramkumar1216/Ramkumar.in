## 🚀 How to Use

### Step 1: Initialize (2 minutes)
Windows: Double-click `setup.bat`  
Mac/Linux: Run `chmod +x setup.sh && ./setup.sh`

### Step 2: Start Development (1 minute)
```bash
# Activate virtual environment
source venv/bin/activate  # Mac/Linux
venv\Scripts\activate     # Windows

# Run server
python manage.py runserver
```

### Step 3: Access & Manage (Ongoing)
- Website: http://localhost:8000
- Admin: http://localhost:8000/admin
- Add content via admin panel

### Step 4: Customize (30-60 minutes)
See CUSTOMIZATION.md for:
- Changing colors
- Updating text/name
- Adding WhatsApp number
- Updating services
- Customizing emails

### Step 5: Deploy (1-2 hours)
See README.md Deployment section for:
- Heroku deployment
- AWS deployment
- Other hosting options
